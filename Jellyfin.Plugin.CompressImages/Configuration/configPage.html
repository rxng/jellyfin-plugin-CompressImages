Compress Images

This plugin compresses oversized images in the Jellyfin People metadata folder using Jellyfin's built-in image encoder (SkiaSharp). No external tools required. It automatically detects the folder location from Jellyfin's internal metadata path. Only images that exceed the configured dimensions or file size are processed.
